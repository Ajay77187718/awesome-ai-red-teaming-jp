# Contributor Covenant 行動規範

## 私たちの誓い

メンバー、コントリビューター、リーダーとして、年齢、体の大きさ、目に見える・見えない障害、民族性、性の特徴、性自認と表現、経験のレベル、学歴、社会経済的地位、国籍、容姿、人種、宗教、または性的アイデンティティと指向に関わりなく、コミュニティへの参加をハラスメントのない体験にすることを誓います。

## 私たちの基準

良好なコミュニティ環境に貢献する行動の例:

- 他の人々への思いやりと親切さ
- 異なる意見、視点、経験の尊重
- 建設的なフィードバックの提供と受容
- 過ちに対する責任の受け入れと謝罪、そして経験からの学び
- 個人だけでなくコミュニティ全体にとって最善なことへの注力

容認できない行動の例:

- 性的な言葉や画像の使用、あらゆる種類の性的注目やアプローチ
- 荒らし、侮辱的・軽蔑的なコメント、個人的または政治的攻撃
- 公的または私的なハラスメント
- 物理的住所やメールアドレスなど、他の人の個人情報の明示的な許可なしの公開
- 専門的な場面で不適切と合理的に判断されるその他の行為

## 執行責任

コミュニティリーダーは、許容される行動の基準を明確にし、執行する責任を持ちます。不適切、脅迫的、攻撃的、または有害とみなされる行動に対して、適切かつ公正な是正措置を講じます。

## 適用範囲

この行動規範は、コミュニティのすべての空間に適用されます。また、個人がコミュニティを公的に代表する場合にも適用されます。

## 執行

虐待的、ハラスメント的、またはその他許容できない行動の事例は、プロジェクトメンテナーに報告してください。すべての苦情は迅速かつ公正に審査・調査されます。
